# Bootcamp Personal Project

> **Unity 기반 포스트 아포칼립스 생존 액션 프로젝트**

부트캠프 과정에서 진행한 개인 프로젝트입니다.  
생존, 전투, 루팅, 제작(크래프팅) 요소를 중심으로 게임플레이를 구현했습니다.

---

## 프로젝트 소개

플레이어는 폐허가 된 세계에서 자원을 수집하고, 적과 전투하며,  
필요한 장비와 시설을 제작해 더 오래 생존해야 합니다.

### 핵심 키워드
- 전투 (근접 공격/피격)
- 인벤토리 & 장비 슬롯
- 월드 아이템 루팅
- 재료 수집 및 제작
- 시설(예: 전력/수자원) 배치

---

## 개발 환경

- **Engine**: Unity
- **Language**: C#
- **Platform**: PC (예정/기준)

---

## 주요 기능

- 플레이어 이동 및 입력 시스템
- 좀비 적 AI 및 전투 상호작용
- 아이템 드롭/획득 시스템
- 인벤토리, 장비, 보관(스태시) UI
- 제작 재료 기반의 아이템 제작 로직

---

## 폴더 구조 (요약)

```text
Assets/
 ├─ 02.Scripts        # 게임 로직 스크립트
 ├─ 04.Prefabs        # 플레이어/적/UI/아이템 프리팹
 ├─ RPGPack           # 타일맵/환경 리소스
 └─ ...               # 이펙트/아이콘/외부 에셋
```

---

## 실행 방법

1. Unity Hub에서 이 프로젝트 폴더를 엽니다.
2. 프로젝트에 맞는 Unity 버전으로 에디터를 실행합니다.
3. 메인 씬을 열고 Play 버튼으로 테스트합니다.
